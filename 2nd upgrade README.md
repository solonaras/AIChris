## 🧠 Core Architecture

### `main.py` – Central Interface
- 🖥 **Multi-modal GUI**: Built with PyQt5, showing real-time mind activity
- 🗣 **Voice Interaction**: Uses Whisper for speech recognition and Edge TTS for speech
- 🧍 **Live Avatar**: Lip-syncing and animated expressions
- 📈 **Monitoring Dashboard**: Displays AI’s current mood, mental health, goals, and thoughts

### `aichris_mind.py` – Cognitive Core
- 🧠 **Ollama AI Integration**: Context-aware natural language response engine
- 🧵 **Memory Layers**: Long/short-term memory with automatic summarization
- 📊 **Performance Feedback**: Tracks response quality and system metrics

## 🚀 Getting Started

### 🛠 Requirements
- Python 3.10+
- PyQt5
- OpenAI Whisper or local transcription
- Edge TTS
- SQLite
- Ollama (for LLM integration)
- FFmpeg (for audio)

### 📦 Installation
```bash
git clone https://github.com/solonaras/AIChris.git
cd AIChris
pip install -r requirements.txt
python main.py
````
